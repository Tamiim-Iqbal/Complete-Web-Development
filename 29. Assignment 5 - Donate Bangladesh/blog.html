What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing developers to interact with and manipulate the content, structure, and style of a website dynamically using languages like JavaScript.

How do you select an element from the DOM?

getElementById(): By ID.
getElementsByClassName(): By class name.
getElementsByTagName(): By tag name.
querySelector(): First matching CSS selector.
querySelectorAll(): All matching CSS selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that involves using a single event listener on a parent element to manage events for multiple child elements. Instead of adding separate listeners to each child, you attach a listener to a common ancestor and utilize the event's propagation up the DOM tree to handle events for all child elements.

How do you manipulate an element's attributes and styles using the DOM?

Attributes:
Get an attribute: element.getAttribute("attributeName")
Set an attribute: element.setAttribute("attributeName", "value")
Remove an attribute: element.removeAttribute("attributeName")
Styles:
Get a style property: element.style.propertyName
Set a style property: element.style.propertyName = "value"
Remove a style property: element.style.removeProperty("propertyName")
